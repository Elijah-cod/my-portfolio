01 · AI systems Python · Retrieval · Graph context

Hybrid RAG Engine

A retrieval architecture that combines semantic relevance with relationship-aware context, making model responses more grounded than a vector-only search pipeline.

The problem
Similarity alone can retrieve related text while missing the relationships that explain it.
The decision
Blend vector signals with graph traversal, then shape a traceable context package.

Anime Tracker

A media tracking product that turns third-party data and changing progress states into an interface that stays fast, legible, and calm.

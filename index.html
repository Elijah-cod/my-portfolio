Code Walkthrough

Auth + retrieval routing 
async def resolve_workspace_query(user, payload):
    if not permissions.can_search(user, payload.workspace_id):
        raise ForbiddenError()

    matches = retriever.rank(
        query=payload.query,
        vectors="workspace_index",
        graph_scope=payload.workspace_id,
    )

    context = graph.expand(matches[:3], depth=2)
    return llm.answer(query=payload.query, context=context)

The important decision here is that model access sits behind explicit permission checks and scoped retrieval, not after the model call. Guard rails belong earlier in the request path.
